專案摘要

這一段直接把 README 的重點整理在頁面上，讀者不用離開展示頁也能看懂專案背景與安全原則。

這個 repo 是什麼

它是一份完整的 ESP32 專案集合，涵蓋讀表、顯示、MQTT 上報與控制功能。

它怎麼演進

從最初的讀表與顯示版本，逐步加入 MQTT、relay、SG90 與感測器整合。

安全做法

真實 Wi-Fi 與 MQTT 帳密都不會放進 repo，請使用 `secrets.example.h` 當作安全範本。

架構總覽

韌體目錄已按照版本整理，讓開發順序、功能分工與檔案結構都一目了然。

Meter Reading

最早的可運作版本，先把電表協定與資料流打通，確保能穩定讀到數值。

Display

透過 OLED 顯示即時資訊，讓設備本機就能直接看到目前狀態。

Connectivity

把量測值封裝成 JSON 後送到 MQTT broker，讓資料能被外部系統接手。
