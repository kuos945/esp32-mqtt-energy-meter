專案架構

repo 已把原本的韌體版本整理成可瀏覽的 firmware 結構，並把各版本的責任劃分清楚，方便後續維護與展示。

Meter Reading

以 `Serial2` 接收電表 frame，負責解析電壓、電流、功率、累積電量與功率因數等欄位。

Display

使用 OLED 與 U8g2 呈現即時資訊，適合在設備本機直接看見運作狀態。

Connectivity

透過 Wi-Fi 與 MQTT 將資料送到 broker，並支援 relay 與 SG90 延伸控制。

韌體版本

這些版本對應 repo 中的 firmware 目錄，適合作為展示頁上的導覽入口。

01

Smart Meter

基礎讀表版本，先把 meter protocol 與資料流打通。

02

OLED Dashboard

在 128x64 OLED 上顯示即時數值與狀態。

03

MQTT Telemetry

把量測值封裝成 JSON 後送到 MQTT broker。

04

Relay Control

加入 MQTT 指令控制繼電器開關。

05

SG90 Expansion

延伸支援 SG90 伺服控制，讓專案更完整。

06

Sensor Fusion

加入額外感測器整合，示範更進階的擴充方式。
